# Suraksha-Setu

Suraksha-Setu is a simple Android-based emergency safety application developed using Java, XML, and Android Studio.

The application is designed to help users quickly send emergency SOS messages to trusted family members during unsafe situations. With a single button press, the app automatically sends an emergency SMS alert to a predefined contact number.

## Features
- One-click SOS emergency button
- Automatic SMS alert system
- Simple and user-friendly interface
- Emergency communication support
- Lightweight Android application

## Technologies Used
- Java
- XML
- Android Studio
- Groovy Gradle

## Purpose
This project was developed as part of Android Fundamentals learning to understand Android application structure, UI design, permissions, and basic mobile app functionality.
